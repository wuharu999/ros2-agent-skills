# ros2 agent skills

Reusable Codex skill documents for ROS 2 robotic arm work. This repository packages implementation, debugging, teaching, and architecture references so an agent can route ROS 2 requests to the right workflow and subsystem guidance with minimal context.

It includes process guides, subsystem patterns, and project structure references covering launch files, URDF, `ros2_control`, MoveIt, Gazebo, CMake, and Python packaging.
